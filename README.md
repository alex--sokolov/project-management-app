Final Task React

# 1. clone repository

```javascript
git clone https://github.com/alex--sokolov/project-management-app.git
```

# 2. switch to custom-app-state branch

```javascript
git checkout "develop"
```

# 3. install dependencies

```javascript
yarn install
```

# 4. start project

```javascript
yarn start
```
